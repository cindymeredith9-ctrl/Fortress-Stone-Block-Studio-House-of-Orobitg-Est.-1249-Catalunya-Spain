7 CONSTRUCTION PHASES:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Phase 1: Foundation & Floor System (2-3 weeks)
• Site preparation
• Footer excavation
• Rebar placement
• Concrete pour
• Plumbing rough-in BEFORE slab
• Electrical conduit placement

Phase 2: Fortress Stone™ Wall Construction (4-6 weeks)
• First course alignment (CRITICAL!)
• Vertical rebar placement
• Bond beams every 4 courses
• Window & door rough openings
• Electrical chases & boxes
• Plumbing penetrations

Phase 3: Catalunya Heritage™ Wooden Beams (1-2 weeks)
• Beam sizing: 8×8, 10×10, or 12×12
• Family coat of arms carving
• Steel bracket installation
• Structural connections

Phase 4: Roof System Installation (2-3 weeks)
• Roof truss/rafter installation
• Roof sheathing
• Underlayment & waterproofing
• Roof tile/shingle installation

Phase 5: Electrical System Integration (1-2 weeks)
• Main service panel (200A minimum)
• Wiring through block chases
• Outlet boxes in walls
• GFCI in wet areas

Phase 6: Plumbing System Integration (1-2 weeks)
• Main water line connection
• PEX or copper distribution
• Drain/waste/vent (DWV) system
• Fixture installation

Phase 7: HVAC System Integration (1-2 weeks)
• Unit sizing calculation
• Equipment installation
• Ductwork through ceiling
• System testing & balancing

CRITICAL: WALL-TO-ROOF CONNECTION
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
The guide includes detailed instructions on THE MOST CRITICAL structural 
connection: tying the walls to the roof!

BOND BEAM SPECIFICATIONS:
• Height: 8-12 inches (full block height)
• Reinforcement: (4) #4 rebar continuous
• Anchor bolts: 5/8" diameter, embedded 7", spaced 4-6 ft
• Concrete: 3,000 PSI minimum
• Cure time: 7 days minimum before roof load

3 ATTACHMENT METHODS:
1. Direct Anchor (for lightweight roofs)
2. Catalunya Heritage™ Beam System (premium)
3. Engineered Truss System (modern)
